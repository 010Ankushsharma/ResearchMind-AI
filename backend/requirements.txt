# only uses `crewai.tools.BaseTool` (tools/web_search_tool.py,

# crewai>=0.1xx uses litellm internally for all LLM calls and no longer
# depends on langchain at all; pinning langchain alongside an old
# crewai<0.70 (which DOES require langchain<0.3) is what made earlier
# versions of this file fail to install — see agents/llm_provider.py.
